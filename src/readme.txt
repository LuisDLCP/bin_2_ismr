This script converts a compressed binary file into an
ISMR file

1) INPUT FILES:
             ~/Input_data/*21_.gz 
2) OUTPUT FILES:
             ~/Output_data/*.ismr

HOW TO USE IT?
Normal use
  1) Move the input files to the folder
            ~/Input_data/
  2) Execute the script bin2ismr.sh
  3) The output files will be saved in:
            ~/Output_files/

Efficient use
  1) Change the REMOTE_PATH in the script bin2ismr_aux.sh
  2) Execute the previous script 
  3) Output files will be saved in:
            ~/Output_files/
